# SkillUp 2027: The Forest Ascension 🌳🚀

A high-performance, gamified life-management dashboard designed to accelerate the journey toward becoming a Computer Scientist by 2027. This ecosystem balances disciplined academic preparation with career-focused mission tracking.

---

## 🗺️ The Gamified Workflow: Forest Ascension

The project is built on a **Duolingo-style roadmap** where growth is visual and gated by progress.

### 1. Beginner's Grove (Level 1 Foundation)
- **Status:** Active / Open.
- **Islands:** Beginner's Grove, Fundamentals Creek, Logic Lake.
- **Workflow:** Daily missions focus on core DSA, CS Fundamentals, and Mathematical logic.
- **Access Logic:** Level 2 is strictly locked until a specific number of Level 1 days/XP are mastered.

### 2. The Hidden Forest (Level 2 & Beyond)
- **Status:** Locked (Requires Level 1 Completion).
- **Focus:** Advanced system design, specialized engineering tracks, and high-level problem solving.

## 🧠 Specialized Hubs

The dashboard features four high-energy portals for different life sectors:

### 🎮 Exam Mastery Mode
A specialized **3-Phase Preparation System** for intense academic cycles:
- **Phase 1: Foundation (Apr 9–15):** 2 subjects per day (one heavy, one light) to build core concepts.
- **Phase 2: Strengthening (Apr 16–20):** Shift from learning to solving with subject-wise MCQ blocks.
- **Phase 3: Exam Mode (Apr 21–24):** Pure simulation with full syllabus mock tests and revision.

### 📊 Data Analyst Dashboard
Focused tracking for specialized data skills:
- **Tracks:** SQL, Advanced Excel, Python for Data, and PowerBI.
- **Metrics:** Daily progress bars and specific topic mastery tags.

### 🚀 Upcoming Missions (Hackathon Hub)
A scalable portal for career milestones:
- Linked directly to a **Global Missions Page**.
- Visual tracking of upcoming Hackathons, Internship deadlines, and Engineering contests.

### 📅 Life Calendar
A clean, glassmorphic UI for long-term planning, event reminders, and life-event tracking, integrated seamlessly into the main HUD.

## ⚙️ The Engine

Behind the visual rewards lies a powerful productivity engine:

- **Dynamic Pomodoro Utility:** Replaces static timers with an adaptive budget system. Automatically sequences **25-minute work blocks** followed by **10-minute breaks** based on the total session time provided by the user.
- **XP & Leveling:** Earn +20 XP per focus block and +50 XP for daily mission completion. Ranks evolve from *Explorer* to *Master*.
- **Energy-Adaptive Themes:** Three distinct UI modes (Min, Norm, High) that adjust visual intensity based on your current focus requirements.
- **LocalState Persistence:** Zero accounts needed. All data (missions, XP, streaks, and hub progress) is securely persisted in your browser's `localStorage`.
